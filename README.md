### What does this project includes?
	Loot generation (container)
	Data driven items/container
	Drag n drop widgets
	InventoryComponent for actors
	Item usage
	Moveable widgets
	Split item
	Pick ups (signle item no inventory)

Currently does not support replication(multiplayer)
